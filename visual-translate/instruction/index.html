🎯 Visual Translator là gì?

Visual Translator là một tiện ích mở rộng (extension) cho trình duyệt giúp bạn dịch văn bản từ hình ảnh, ảnh chụp màn hình và âm thanh bằng công nghệ OCR (nhận dạng ký tự quang học) và chuyển đổi văn bản thành giọng nói.

📦 Cài đặt

Bước 1: Cài đặt extension

  • Tải file Visual Translator từ Chrome Web Store
  • Nhấp vào "Thêm vào Chrome"
  • Xác nhận cài đặt khi được yêu cầu

Bước 2: Cấp quyền truy cập

  • Cho phép extension truy cập vào các trang web
  • Cho phép quyền chụp màn hình
  • Cho phép quyền truy cập âm thanh (nếu cần)

⚙️ Cấu hình ban đầu

Thiết lập API dịch thuật

  • Mở extension từ icon trên thanh công cụ
  • Đi đến mục "Cài đặt" hoặc "Settings"
  • Nhập API key cho dịch vụ dịch thuật:
    • Google Translate API: Nhập API key của bạn
    • OpenAI API: Nhập API key cho ChatGPT
  • Chọn ngôn ngữ nguồn và ngôn ngữ đích mặc định
💡 Mẹo: Bạn có thể thay đổi cài đặt bất cứ lúc nào từ menu cài đặt của extension.

🚀 Các tính năng chính

📸 Dịch từ ảnh chụp màn hình

  • Chụp màn hình bất kỳ khu vực nào
  • OCR tự động nhận dạng văn bản
  • Dịch văn bản sang ngôn ngữ đã chọn

🖼️ Dịch từ hình ảnh

  • Tải lên hình ảnh từ máy tính
  • Nhận dạng văn bản trong ảnh
  • Dịch và hiển thị kết quả

🎤 Dịch từ âm thanh

  • Ghi âm giọng nói
  • Chuyển đổi giọng nói thành văn bản
  • Dịch văn bản sang ngôn ngữ khác

📝 Dịch văn bản trực tiếp

  • Nhập hoặc dán văn bản cần dịch
  • Hỗ trợ nhiều cặp ngôn ngữ
  • Lưu lịch sử dịch

🎮 Cách sử dụng các tính năng

📸 Chụp và dịch màn hình

Cách 1: Dùng phím tắt

  • Nhấn Alt + S để chụp màn hình
  • Chọn khu vực cần chụp bằng chuột
  • Chờ extension xử lý và hiển thị kết quả dịch

Cách 2: Dùng menu chuột phải

  • Chuột phải vào trang web
  • Chọn "Visual Translator" → "Chụp màn hình"
  • Chọn khu vực và chờ kết quả

🖼️ Dịch từ hình ảnh

  • Mở extension từ icon trên thanh công cụ
  • Chọn tab "Hình ảnh" hoặc "Image"
  • Nhấp vào "Chọn hình ảnh" hoặc kéo thả file
  • Chờ OCR xử lý và hiển thị kết quả dịch

🎤 Ghi âm và dịch

  • Mở extension và chọn tab "Âm thanh" hoặc "Audio"
  • Nhấp vào nút "Bắt đầu ghi âm"
  • Nói rõ ràng vào micro
  • Nhấp "Dừng ghi âm" và chờ xử lý
  • Kết quả sẽ được hiển thị và dịch

📝 Dịch văn bản trực tiếp

  • Mở extension
  • Chọn tab "Văn bản" hoặc "Text"
  • Nhập hoặc dán văn bản cần dịch
  • Chọn cặp ngôn ngữ
  • Nhấp "Dịch" để xem kết quả

🔧 Các chế độ hoạt động

🎯 Popup Mode

  • Nhấp vào icon extension
  • Hiển thị cửa sổ nhỏ nhanh
  • Thích hợp cho dịch nhanh

📋 Side Panel Mode

  • Mở cửa sổ bên cạnh trình duyệt
  • Làm việc song song với web
  • Thích hợp cho dịch dài

⚡ Phím tắt và mẹo

Phím tắt hữu ích:

  • Alt + S - Chụp màn hình nhanh
  • Ctrl/Cmd + Shift + V - Mở extension
💡 Mẹo chuyên nghiệp:
  • Sử dụng Side Panel để dịch nhiều văn bản cùng lúc
  • Lưu các cặp ngôn ngữ hay dùng để truy cập nhanh
  • Kiểm tra lại API key nếu dịch không hoạt động
  • Sử dụng văn bản rõ ràng để OCR chính xác hơn

⚠️ Lưu ý quan trọng

🔒 Bảo mật và quyền riêng tư:

  • Tất cả dữ liệu được xử lý cục bộ trên máy của bạn
  • Extension không thu thập thông tin cá nhân
  • API key được lưu trữ an toàn trong trình duyệt
  • Không gửi dữ liệu đến máy chủ của chúng tôi

📝 Yêu cầu hệ thống:

  • Kết nối internet để gọi API dịch thuật
  • API key hợp lệ cho dịch vụ đã chọn
  • Microphone (cho tính năng ghi âm)
  • Chrome/Edge phiên bản mới nhất

🛠️ Gỡ rối sự cố

Vấn đề thường gặp:

  • Dịch không hoạt động: Kiểm tra API key và kết nối internet
  • OCR không chính xác: Đảm bảo hình ảnh rõ nét, văn bản đủ lớn
  • Không ghi được âm thanh: Kiểm tra quyền micro trong cài đặt trình duyệt
  • Extension không hoạt động: Tải lại trang hoặc khởi động lại trình duyệt
